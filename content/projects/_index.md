These are projects I've been working on recently.  I use these projects as artists use sketches: to explore, capture, or describe a technique or idea or technology.  I don't finish many.

Each is mostly just a proof-of-concept, but longer-running projects will accumulate a backlog, and I generally set up CI/CD processes, use some form of Test-Driven Design, etc... just because I've found most aspects of modern best practices in professional software development to be useful in my personal projects as well.
