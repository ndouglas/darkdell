Oldtown is a Rust library and application for implementing neural networks. It provides tools and utilities for building and training neural networks, as well as an easy-to-use application for deploying them in real-world settings. The library is designed to be flexible and extensible, and supports a wide range of neural network architectures and training techniques.

Oldtown is intended for use by developers who want to create powerful AI systems for a wide range of applications, from image recognition and natural language processing to game AI and robotics. The library's clean and intuitive interface, combined with its powerful performance and flexibility, make it a versatile tool for anyone interested in exploring the possibilities of neural networks.
