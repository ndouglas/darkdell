The library is designed to be highly customizable and versatile, allowing users to create a wide range of systems, from simple monotheistic religions to complex pantheons of deities and supernatural creatures.

## Summary

Raventree is a Rust library for procedurally generating supernatural systems, religions, and magical powers. The library provides users with a way to create complex, detailed systems that include elements such as monotheistic religions, pantheons of deities, supernatural creatures, magic, and more.

The library is built using Rust and incorporates a range of algorithms and techniques from fields such as anthropology, mythology, and folklore. It is designed to be highly customizable and extendable, so that users can easily add their own algorithms and features to the system.

### Naming

Raventree is the ancestral seat of House Blackwood in the A Song of Ice and Fire series, and in the books, it is described as a place surrounded by a dark forest filled with ravens. The imagery of the raven is often associated with magic, the supernatural, and the afterlife, making "Raventree" an appropriate name for this project.

## Motivations

The motivation for this project comes from a deep interest in mythology, folklore, and the supernatural. The goal of Raventree is to provide a tool that makes it easier for other developers to explore these topics and create their own supernatural systems and magical powers that are richly detailed and unique.

## Challenges

One of the main challenges of this project is the sheer complexity of the task at hand. Creating a system that can generate realistic and diverse supernatural systems and magical powers, with all the necessary cultural and historical context, is a daunting task that requires a deep understanding of a range of disciplines.

To tackle this challenge, we plan to start with the basics and build up from there, gradually adding more complex features and refining the algorithms as we go. We will also be consulting with experts in the fields of anthropology, mythology, and folklore to ensure that our models and algorithms are as accurate and up-to-date as possible.
